# ✏️Name<br>
  
  Chatn<br>
  
# 📚Feachure<br>
  
  フレームワークはRuby on Rails、 バックエンドはRuby、<br>
  フロントエンドはJavaScriptでチャットアプリを作成しました。<br>
  MySQL(development)、postgreSQL(production)を使用しています。<br>
  ActionCableを用いることにより、<br>
  低コストかつ、リアルタイムでメッセージを送受信<br>
  することが可能です。<br>
  
# 💻Requirement<br>

  ruby 2.5.8<br>
  Ruby on Rails 6.0.3<br>
